# Image and Animation using Python

This repository contains a Jupyter Notebook demonstrating image processing
and basic animation techniques using Python. The project is designed for
learning, experimentation, and showcasing practical Python skills.

#  Features
- Image loading and visualization
- Basic image manipulation
- Simple animation creation
- Clean and well-structured Jupyter Notebook
